# Color Palette Generator

A modern, minimalist web application to generate random color palettes with a single click. Designed for developers and designers looking for quick color inspiration.

## Features

- **Instant Generation**: Create a fresh 5-color palette instantly by clicking the "Generate" button.
- **Click-to-Copy**: Easily copy HEX codes to your clipboard by clicking either the color box or the copy icon.
- **Visual Feedback**: Smooth CSS transitions and a checkmark icon confirmation when a color is successfully copied.
- **Fully Responsive**: Optimized layout using CSS Grid and Flexbox for desktop, tablet, and mobile devices.
- **Modern UI**: Clean interface featuring *Google Sans* typography and elegant background gradients.

## Built With

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, Grid, Flexbox, and CSS Animations.
- **JavaScript (ES6+)**: Random color logic and Clipboard API integration.
- **Fonts**: Google Sans for a premium feel.
- **Icons**: Font Awesome (for the copy/success states).
